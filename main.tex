% Abstract

\makeabstracttitle

Modern hardware increasingly provides powerful security mechanisms that operate partially outside the operating system’s traditional authority.
At the same time, the long-standing OS protection model, built around coarse-grained process isolation, struggles to address the security and performance demands of modern software.
This divergence leads to incoherent protection models, where hardware and software enforce security independently, leaving gaps that attackers can exploit.

This dissertation argues that effective system security requires coherent protection: hardware-assisted protection models must align systematically with those implemented by the OS.
Through this lens, the dissertation examines two emerging challenges: (1) hardware-assisted in-process isolation, and (2) side-channel mitigation in VM-level trusted execution environments.
To address these challenges, it presents two systems, \capacity and \incognitos, which retrofit OS security mechanisms to modern hardware features without sacrificing compatibility.

\capacity demonstrates how intra-process isolation domains established through ARM hardware primitives—such as Pointer Authentication and Memory Tagging—can be made coherent with legacy OS abstractions by transforming resource references into cryptographically authenticated capabilities.

\incognitos addresses the challenge of aligning OS-level behavior with hardware-enforced VM isolation. 
It redesigns OS capabilities \dash\ scheduling, and memory management \dash\ to mitigate side-channel leakage in confidential VMs while preserving compatibility and performance.

Together, these systems provide practical blueprints for reconciling modern hardware security primitives with long-standing OS protection models.
They demonstrate that, through principled retrofitting, contemporary systems can leverage new hardware capabilities without sacrificing performance, compatibility, or security.

The ideas from this dissertation contribute to the vision of secure computing infrastructures where hardware and software collaborate seamlessly to defend against current and emerging threats.




\textbf{Keywords:} Security, Operating System, Protection, Side channels
